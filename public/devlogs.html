Technical Details

Backend
Node.js with Express server, hosted on Vercel. Uses @tomphttp/bare-server for proxy functionality.
Frontend
Vanilla HTML/CSS/JavaScript with Lucide icons. Responsive design with modern CSS features.
Security
Ultraviolet service worker for secure proxying. Implements proper encoding and caching strategies.
Performance
Optimized service worker caching, lazy loading, and efficient resource management.
