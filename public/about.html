Security & Privacy First

Your privacy and security are our top priorities. Here's how we protect you:

Isolated Browsing

All proxied content runs in isolated iframes with no direct access to your browser's main context, ensuring complete separation from your personal data.

Zero Data Logging

We don't log, store, or track any of your browsing activity. All proxying is done in real-time with no persistence whatsoever.

Client-Side Processing

URL encoding and request rewriting happens entirely in your browser using service workers, keeping your destinations private.

Minimal Server Exposure

The server only proxies requests and serves static files. No user data is stored, processed, or accessible server-side.

How Ambient Works

A transparent look at our proxying architecture

01

URL Input

You enter a website URL in the search bar on the home page.

02

Service Worker Registration

Ultraviolet service worker is registered with a specific scope to intercept all subsequent requests.

03

URL Encoding

The URL is encoded using XOR cipher to obfuscate the destination before transmission.

04

Session Storage

Encoded URL is temporarily stored in sessionStorage for the proxy page to access securely.

05

Page Navigation

You're redirected to the search page where an iframe loads the encoded URL seamlessly.

06

Request Interception

Service worker intercepts all iframe requests and rewrites them for secure proxying.

07

Bare Server Routing

Requests are routed through the Bare server at the /bare/ endpoint on the backend.

08

Response Processing

Server fetches the target website and returns the content, which gets rewritten by the service worker.

09

Content Injection

Ultraviolet injects client-side scripts to handle dynamic content and maintain full functionality.

10

Isolated Rendering

Final proxied website renders in the iframe with full functionality but completely isolated from your main browser context.
